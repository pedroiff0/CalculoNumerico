Tutorial — calcnum

Este tutorial mostra como executar o arquivo principal calcnum.py de forma não interativa usando arquivos de entrada em tests/inputs/ e como reproduzir os exemplos usados nos testes de integração.

Executando um exemplo (linha de comando)

python calcnum.py < tests/inputs/inputSistemas.txt

O arquivo acima contém as entradas simuladas para a opção “Sistemas Lineares”. Abaixo mostramos um exemplo do conteúdo do arquivo de entrada.

2
1
2
3 1
1 2
9 8
0
0

Outros arquivos de input (exemplos)

  • tests/inputs/inputAjustes.txt

  • tests/inputs/inputBases.txt

  • tests/inputs/inputEDOS_calcnum.txt

  • tests/inputs/inputIntegracoes_calcnum.txt

  • tests/inputs/inputInterpolacoes.txt

  • tests/inputs/inputRaizes.txt

Dica: execute com python calcnum.py < tests/inputs/<arquivo> para validar comportamento de menu de forma repetível e adequada para CI.
