Métodos de Raízes

Funções do módulo codigos.raizes para cálculo de raízes (Bisseção, Newton, Secante).

Documentação

codigos.raizes.bissecao(func_str, a, b, tol, max_iter)[código-fonte]

Método da bisseção para encontrar raiz de uma função dada por string.

Parâmetros:

  • func_str (str) – Expressão Python de f(x) (ex.: 'x**2 - 4').

  • a (float) – Intervalo inicial [a, b] com sinais opostos.

  • b (float) – Intervalo inicial [a, b] com sinais opostos.

  • tol (float) – Tolerância para critério de parada.

  • max_iter (int) – Número máximo de iterações.

Retorna:

Tupla (raiz_aproximada, n_iter) ou (None, 0) se os sinais em a/b não forem opostos.

Tipo de retorno:

(float, int)

codigos.raizes.newton(func_str, x0, tol, max_iter)[código-fonte]

Método de Newton-Raphson para encontrar raiz de função dada por string.

Parâmetros:

  • func_str (str) – Expressão Python de f(x).

  • x0 (float) – Chute inicial.

  • tol (float) – Tolerância para critério de parada.

  • max_iter (int) – Número máximo de iterações.

Retorna:

Tupla (raiz_aproximada, n_iter) ou (None, n_iter) se houver erro.

Tipo de retorno:

(float, int)

codigos.raizes.secante(func_str, x0, x1, tol, max_iter)[código-fonte]

Método da secante para encontrar raiz de função dada por string.

Parâmetros:

  • func_str (str) – Expressão Python de f(x).

  • x0 (float) – Dois chutes iniciais.

  • x1 (float) – Dois chutes iniciais.

  • tol (float) – Tolerância para critério de parada.

  • max_iter (int) – Número máximo de iterações.

Retorna:

Tupla (raiz_aproximada, n_iter) ou (None, n_iter) em caso de erro.

Tipo de retorno:

(float, int)

codigos.raizes.pedir_dados_raizes(metodo=None)[código-fonte]

Lê os dados necessários para o método de raízes.

Se metodo for None, comporta-se como modo interativo completo (executa o método e plota o resultado). Se metodo for uma das strings 'bissecao', 'newton' ou 'secante', apenas lê os parâmetros e retorna uma tupla (func_str, tol, max_iter, params) para uso por menus externos (compatibilidade com menu_raizes).

Exemplo de Uso

1"""Exemplo simples de uso: encontrar raiz com bisseção"""
2
3from codigos import raizes
4
5if __name__ == "__main__":
6    raiz, iters = raizes.bissecao("x**2 - 4", 0, 3, 1e-8, 100)
7    print(f"Raiz encontrada: {raiz} (iteração {iters})")

 1import math
 2from codigos import raizes
 3import pytest

 5def test_bissecao_basic():
 6    raiz, iters = raizes.bissecao('x**2 - 4', 0.0, 3.0, 1e-8, 100)
 7    assert raiz is not None

11def test_bissecao_invalid_interval():
12    raiz, iters = raizes.bissecao('x**2 + 1', -1.0, 1.0, 1e-8, 10)
13    assert raiz is None
14    assert iters == 0
15
16def test_newton_basic():
17    raiz, iters = raizes.newton('x**2 - 2', 1.5, 1e-12, 50)
18    assert raiz is not None
19    assert abs(raiz - math.sqrt(2.0)) < 1e-10
20
21def test_newton_derivative_zero():
22    # f(x) = x^3 at x=0 has derivative = 0; finite-difference may yield tiny derivative,
23    # but the method should produce a root (x=0) or None; accept 0.0 as valid root here.
24    raiz, iters = raizes.newton('x**3', 0.0, 1e-12, 10)
25    assert raiz is not None
26    assert abs(raiz - 0.0) < 1e-12
27
28def test_secante_basic():
29    raiz, iters = raizes.secante('x**3 - 0.5', 0.0, 1.0, 1e-10, 100)
30    assert raiz is not None
31    # verify residual is small
32    fval = (raiz**3 - 0.5)
33    assert abs(fval) < 1e-8
34
35def test_pedir_dados_raizes_for_menu(monkeypatch):
36    # Simulate inputs for bissecao: func, tol, max_iter, a, b
37    inputs = iter(['x**2 - 4', '1e-8', '100', '0', '3'])
38    monkeypatch.setattr('builtins.input', lambda prompt='': next(inputs))
39    func_str, tol, max_iter, params = raizes.pedir_dados_raizes('bissecao')
40    assert func_str == 'x**2 - 4'
41    assert abs(float(tol) - 1e-8) < 1e-20
42    assert int(max_iter) == 100
43    assert params == (0.0, 3.0)
